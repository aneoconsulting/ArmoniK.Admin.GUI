Understanding and using the application

This page allows you to understand all the functionalities present in the interface in order to use it at best.

Display data

Different data, such as sessions, tasks, are displayed in different tables within the application.

These tables allow you to view, sort and filter data. A column is sortable as soon as the icon to filter is visible. To sort a column, click on the name of the column. To filter a column, click on the icon .

Navigate between pages

The navigation within the application is done in different ways.

  • Via the dashboard. The latter lists all the applications within ArmoniK. It is then possible to navigate within the sessions of an application and then within the tasks of a session.
  • Via the tabs. These tabs allow you to quickly switch from one application to another once you've been there.

Find the errors

It may happen that an error occurs while using the application. This can be shown by the appearance of a red banner or by the display of an error page.

When this happens, it is possible to find out more about the error via the console and the network tab of the browser inspector.
When reporting bugs, it is important to forward this error as well as the version number of the application.

Change the language

The application is available in 2 languages: French and English. To change it, you just have to click on the corresponding button in the the header of the page, on the top right.

The language currently in use is no longer available for selection and its selection and displayed is grayed out color. Also, your selection is kept in memory for your next future navigations within the application.

See the version

The version of the application is displayed in the bottom left navigation bar. It gives information about the version of the application of the application used.

It is important to to forward this version to the development team when a bug is reported in order to facilitate the correction of these the latter.
